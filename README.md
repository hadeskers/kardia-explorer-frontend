# Kardia block explorer

## Requirement
1. Blocks
    - Blocks list
    - Block detail
2. Transactions
    - Transactions list
    - Transaction detail
3. Networks + nodes
    - Nodes list
4. Wallet
    - Create wallet
    - Access wallet
5. Faucet
    - Get testnet KAI tokens

## Development
1. Clone from this project
2. Run `npm install`
3. Run `npm start`

## Test
Run `npm test`

## Build
Run `npm run build`
